**Fit and Fuel**

Fit and Fuel is a desktop application built with Python and Tkinter that enables users to manage their fitness and nutrition in a single, centralized platform. The application is designed to simplify health tracking by providing an intuitive interface for logging workouts, recording meals, and monitoring overall progress.

**Overview**

Many individuals rely on multiple applications to track their health data, which can lead to inefficiency. Fit and Fuel addresses this problem by combining workout tracking and nutrition management into one cohesive system. The application emphasizes ease of use and accessibility.

**Features**
Account creation and login functionality
Personalized user experience
Dashboard
Centralized interface for navigation
Quick access to core features
Log exercises and workout sessions
Record duration, type, and intensity
Meal Tracking
Log daily food intake
Track calorie consumption
Designed to support future integration with AI-based calorie estimation
Progress Monitoring
